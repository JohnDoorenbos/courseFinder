API Documentation

The API for this application allows you to get information from a database containing all of the information that can be found in the course catalog.


Requests are to be made to norsecourse.herokuapp.com/api. By default it returns a dictionary of the entire database.


If you would like to make a more specific request, you can specify what information you would like to recieve as if you were filling out a form.


For example

  • norsecourse.herokuapp.com/api?dept=rel : Returns all of the courses in the religion department.
  • norsecourse.herokuapp.com/api?gen_eds=HB&gen_eds=Quant : Returns a dictionary of all of the courses that satisfy the HB (Human Behavior) or Quant (Quantitative) general education requirements
  • norsecourse.herokuapp.com/api?dept=rel&gen_eds=HB : Returns all of the courses in the religion department that satisfy the HB (Human Behavior) general education requirement.
  • norsecourse.herokuapp.com/api?id=CS%20365 : Returns the course with id CS 365 NOTE: The %20 represents a space.

For a complete reference of searchable parameters see below:

  • title: Search by title keyword (ex: title=Introduction) Note: Case sensitive
  • dept: Search by department name (ex: dept = rel) Note: Case insensitive
  • gen_eds: Search by general education requirements (ex: gen_eds = Quant) Note: you can search for multiple gen_eds by saying gen_eds=HB&gen_eds=HEPT
  • prereqs: Search by prerequisite classes (ex: prereq s=BIO%20151) Note: %20 represents a space
  • id: Search by course id (ex: id=CS%20365) Note: %20 represents a space
